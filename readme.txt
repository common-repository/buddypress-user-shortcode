== Description ==

This Plugin offers a shortcode ([bpmembers]) which returns the number of registred Buddypress users. The Shortcode can be used in posts, pages and sidebar widgets.

= German =
Dieses Plugin gibt &uuml;ber einen Shortcode([bpmembers]), der in Beitr&auml;gen, Seiten und Widgets verwendet werden kann, die Anzahl der registrierten Buddypress Mitglieder zur&uuml;ck.

More informations on the official plugin page: [http://sxss.nw.am](http://sxss.nw.am "sxss.nw.am")

== Installation ==

Upload the plugin to your blog, activate it, done!
